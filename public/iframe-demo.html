Starlight Iframe Embedding Demo

This demo shows how Starlight documentation pages can be embedded in iframes with navigation elements hidden.

How to Use

To embed any Starlight documentation page in an iframe on your own site, simply use this code:

<iframe src="https://your-starlight-site.com/your-page-path?iframe=true"></iframe>

The page will only hide navigation elements when the ?iframe=true parameter is included.
