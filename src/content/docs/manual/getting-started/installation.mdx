Scalekit helps you quickly add enterprise authentication to your B2B SaaS app without engineering overhead. It provides plug-and-play auth solutions like SSO, SCIM, and Social Logins.

![](@/assets/docs/sso-quickstart/how-scalekit-connects.png)

<Steps>
1. [Sign up](https://app.scalekit.com/ws/signup) for a Scalekit account and retrieve your API keys.

2. Add your API credentials to your projectâ€™s environment. For example, create or update your `.env` file as follows:
   <Env />

3. Install the Scalekit SDK for your preferred tech stack using the instructions below:
   <InstallSDK />

</Steps>

### Testing your installation

<Steps>

1. Instantiate the SDK in your project.
   <Tabs>
     <TabItem value="javascript" label="Node.js">
       ```js showLineNumbers=false
       import { Scalekit } from '@scalekit-sdk/node';

       const scalekit = new Scalekit(
          process.env.SCALEKIT_ENVIRONMENT_URL,
          process.env.SCALEKIT_CLIENT_ID,
          process.env.SCALEKIT_CLIENT_SECRET,
        );
       ```
     </TabItem>
     <TabItem value="javascript" label="Python">
       ```py showLineNumbers=false
       from scalekit import ScalekitClient, AuthorizationUrlOptions,CodeAuthenticationOptions

       scalekit = ScalekitClient(
          '<SCALEKIT_ENVIRONMENT_URL>',
          '<SCALEKIT_CLIENT_ID>',
          '<SCALEKIT_CLIENT_SECRET>'
        )
       ```
     </TabItem>
     <TabItem value="go" label="Golang">
       ```go showLineNumbers=false
       import (
          "github.com/scalekit/scalekit-sdk-go"
        )

       scalekitClient := scalekit.NewScalekitClient(
          "<SCALEKIT_ENVIRONMENT_URL>",
          "<SCALEKIT_CLIENT_ID>",
          "<SCALEKIT_CLIENT_SECRET>"
        )
       ```
     </TabItem>
     <TabItem value="java" label="Java">
       ```java showLineNumbers=false
       package com.scalekit;
       import com.scalekit.ScalekitClient;

       ScalekitClient scalekitClient = new ScalekitClient(
          "<SCALEKIT_ENVIRONMENT_URL>",
          "<SCALEKIT_CLIENT_ID>",
          "<SCALEKIT_CLIENT_SECRET>"
        );
       ```
     </TabItem>
   </Tabs>
2. Test your installation listing organizations
   <Tabs>
     <TabItem value="javascript" label="Node.js">
       ```js showLineNumbers=false
       import { Scalekit } from '@scalekit-sdk/node';

    const scalekit = new Scalekit(
      process.env.SCALEKIT_ENVIRONMENT_URL || '',
      process.env.SCALEKIT_CLIENT_ID || '',
      process.env.SCALEKIT_CLIENT_SECRET || ''
    );

    const organizations = await scalekit.organization.listOrganization({});

    console.log(organizations);
   ```
     </TabItem>
   </Tabs>
