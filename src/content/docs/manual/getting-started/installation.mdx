This guide explains how to install and configure the Scalekit SDK to add enterprise authentication features to your application. After completing this guide, you'll be able to:

- Set up your Scalekit development environment
- Install the SDK for your programming language
- Verify your installation works correctly

## Prerequisites

Before you begin, make sure you have:

- A Scalekit account
- Your preferred development environment set up (Node.js, Python, Go, or Java)
- Basic familiarity with your chosen programming language

## Installation Steps

<Steps>
1. Create a Scalekit Account
   - [Sign up](https://app.scalekit.com/ws/signup) for a Scalekit account
   - Navigate to API Settings to retrieve your API keys

2. Configure Environment Variables
   Add your API credentials to your project's environment variables:
   <Env />

3. Install the SDK
   Choose your programming language and follow the installation instructions:
   <InstallSDK />
</Steps>

## Verify Your Installation

Follow these steps to verify that the SDK is installed and configured correctly:

<Steps>
1. Initialize the SDK
   <Tabs>
     <TabItem value="javascript" label="Node.js">
       ```js showLineNumbers=false
       import { Scalekit } from '@scalekit-sdk/node';

       const scalekit = new Scalekit(
          process.env.SCALEKIT_ENVIRONMENT_URL,
          process.env.SCALEKIT_CLIENT_ID,
          process.env.SCALEKIT_CLIENT_SECRET,
        );
       ```
     </TabItem>
     <TabItem value="javascript" label="Python">
       ```py showLineNumbers=false
       from scalekit import ScalekitClient, AuthorizationUrlOptions,CodeAuthenticationOptions

       scalekit = ScalekitClient(
          '<SCALEKIT_ENVIRONMENT_URL>',
          '<SCALEKIT_CLIENT_ID>',
          '<SCALEKIT_CLIENT_SECRET>'
        )
       ```
     </TabItem>
     <TabItem value="go" label="Golang">
       ```go showLineNumbers=false
       import (
          "github.com/scalekit/scalekit-sdk-go"
        )

       scalekitClient := scalekit.NewScalekitClient(
          "<SCALEKIT_ENVIRONMENT_URL>",
          "<SCALEKIT_CLIENT_ID>",
          "<SCALEKIT_CLIENT_SECRET>"
        )
       ```
     </TabItem>
     <TabItem value="java" label="Java">
       ```java showLineNumbers=false
       package com.scalekit;
       import com.scalekit.ScalekitClient;

       ScalekitClient scalekitClient = new ScalekitClient(
          "<SCALEKIT_ENVIRONMENT_URL>",
          "<SCALEKIT_CLIENT_ID>",
          "<SCALEKIT_CLIENT_SECRET>"
        );
       ```
     </TabItem>
   </Tabs>
2. Run a Test Query
   Execute this sample code to list organizations:
   <Tabs>

## Troubleshooting

If you encounter issues during installation:

- Verify your API credentials are correct in your environment variables
- Ensure you're using a supported version of your programming language
- Check that all required dependencies are installed
- Review the [error reference guide](/reference/errors) for specific error messages
