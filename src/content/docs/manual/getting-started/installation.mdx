Scalekit helps you quickly add enterprise authentication to your B2B SaaS app without engineering overhead. It provides plug-and-play auth solutions like SSO, SCIM, and Social Logins.

![](@/assets/docs/sso-quickstart/how-scalekit-connects.png)

<Steps>
1. [Sign up](https://app.scalekit.com/ws/signup) for a Scalekit account and retrieve your API keys.

2. Add your API credentials to your projectâ€™s environment. For example, create or update your `.env` file as follows:

   ```sh showLineNumbers=false
   SCALEKIT_CLIENT_ID='<CLIENT_ID_FROM_SCALEKIT_DASHBOARD>'
   SCALEKIT_CLIENT_SECRET='<SECRET_FROM_SCALEKIT_DASHBOARD>'
   ```
