## Core Features

<CardGrid stagger>
  <Card title="Single Sign-On (SSO)" icon="rocket">
    Support for SAML, OAuth, and OpenID Connect. Let your enterprise customers log in with their existing identity providers.
  </Card>
  <Card title="SCIM Provisioning" icon="random">
    Automatic user provisioning and deprovisioning. Synchronize user accounts with your customer's directory.
  </Card>
  <Card title="Social Logins" icon="codePen">
    Allow users to sign in with Google, GitHub, Microsoft and other popular providers.
  </Card>
  <Card title="Multi-Tenant Ready" icon="setting">
    Built for B2B SaaS applications with multi-tenant architecture.
  </Card>
</CardGrid>
